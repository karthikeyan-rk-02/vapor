Vapor - Pomodoro Focus App

A clean, distraction-free Pomodoro Timer web app designed for deep focus and productivity.
No login. No backend. No distractions. Just focus.

Features
⏱25/5 Pomodoro Timer (customizable)
Minimal glowing orange UI (focus-centric design)
One-click Start / Pause / Reset
Automatic session switching (Focus ↔ Break)
Saves progress using localStorage
Instant load — no setup required
Fully responsive (works on desktop & mobile)
Design Philosophy

Vapor is built with simplicity in mind:

“The less you see, the more you focus.”

Dark UI to reduce eye strain
Glowing orange accent to represent energy
No clutter, no unnecessary features
